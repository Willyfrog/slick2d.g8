
Description
-----------
This is a small extra to make g8 philcali/slick2d template into a
project with intelliJ Idea support.

Requirements
------------
You'll need sbt and scala plugins in the ide.

Installation
------------
install template:  
    $ g8 willyfrog/slick2d
generate idea's project files:
    $ sbt gen-idea
